## 次の作業

SQLiteからクイズ一覧を取得する関数を作る。

例

```python
from database import get_connection

def load_quizzes():
    conn = get_connection()

    rows = conn.execute(
        "SELECT * FROM quizzes"
    ).fetchall()

    conn.close()

    return [dict(row) for row in rows]
```

そして

```python
quiz_list = load_quizzes()
```

へ置換する。
