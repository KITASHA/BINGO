# 🎉 BINGOツール

イベント・懇親会・レクリエーション向けのBINGO・クイズ運営ツールです。

---

## ✨ 主な機能

- 🎲 ビンゴ抽選
- ❓ クイズ抽選
- 📝 クイズの追加・編集・削除
- 🖼️ 問題画像・回答画像対応
- 🔥 フィーバークイズ
- 📥 クイズのインポート（JSON）
- 📤 クイズのエクスポート（JSON）
- 🚫 出題除外設定

---

## 🚀 起動方法

1. `BINGOツール.exe` を起動します。
2. ブラウザが自動で開きます。
3. 表示画面と操作画面を利用して進行してください。

---

## 📥 初期クイズの読み込み

初期クイズを利用する場合は、

1. 管理画面を開く
2. **追加インポート** をクリック
3. `quizzes.json` を選択

これでクイズが追加されます。

---

## 📝 クイズ管理

管理画面から次の操作ができます。

- クイズ追加
- 編集
- 削除
- 出題除外
- フィーバー設定
- JSONインポート
- JSONエクスポート

---

## 📁 配布フォルダ

```
BINGOツール
│
├─ BINGOツール.exe
├─ quizzes.json
└─ _internal
```

`_internal` フォルダは削除・移動しないでください。

---
